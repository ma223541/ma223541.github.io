我的第一篇博客

今天学习了 Markdown，感觉非常方便。

我的学习计划

  1. 学习基础语法
  2. 写一篇测试文章
  3. 发布到博客上

这是一段引用。

简单来说，用Markdown语法创建 .md 文件，就是指用一种**非常简单的“标记符号”**来写文章，然后把它保存成一个以 .md 结尾的文件。

我们可以把它拆解成三部分来理解：

1. 什么是Markdown语法？

Markdown是一种轻量级标记语言。它用一些简单的符号（如 #、*、-）来给文字添加格式，比如标题、列表、加粗等。

它的核心思想是：让你在写作时，手指不离开键盘，就能完成排版。

  • 普通文本：你好，世界。
  • Markdown语法：

你不需要学习复杂软件，就像在记事本里打字一样，只是在需要格式的地方，加几个简单的符号。

2. 什么是 .md 文件？

.md 就是一个文件扩展名，就像你常见的 .txt（记事本）、.doc（Word文档）、.jpg（图片）一样。

  • .txt 文件：纯文本，没有任何格式（不能加粗、不能调字号）。
  • .doc 文件：富文本，包含复杂的格式信息（字体、颜色、对齐等），需要用Word等专用软件打开。
  • .md 文件：纯文本 + Markdown标记。它本质上还是纯文本，但因为有那些特殊的标记符号，通过专用软件（如VS Code、Typora）或网站（如GitHub）打开后，就能自动渲染成美观的排版。

3. “创建 .md 文件”具体怎么做？

这和你创建一个 .txt 文件一样简单，有几种方式：

  • 方法一：用代码编辑器（推荐）
    使用 VS Code、Sublime Text、Typora 这类编辑器。

    1. 新建一个文件。
    2. 直接用Markdown语法写内容。
    3. 保存时，将文件名后缀设为 .md，比如 my-first-blog.md。

  • 方法二：用最简单的记事本

    1. 在电脑上新建一个 .txt 文件，比如 文章.txt。
    2. 用Markdown语法写内容。
    3. 保存后，将文件名从 文章.txt 重命名为 文章.md。（如果系统没显示扩展名，需要先设置显示）。

  • 方法三：在Hexo等框架中直接创建
    当你运行 hexo new "我的文章" 这样的命令时，Hexo会自动在 source/_posts 文件夹里帮你生成一个 .md 文件，比如 我的文章.md。你只需要打开它，在预置的框架里填写内容即可。

回到你的问题：在搭建博客的上下文中

当教程说“在 source/_posts 文件夹里，用Markdown语法创建 .md 文件”时，它的完整意思就是：

  1. 找到你博客项目里的 source/_posts 文件夹。
  2. 新建一个文件，名字可以叫 2024-10-27-my-first-post.md。
  3. 用Markdown的规则（比如用 # 表示标题，* 表示列表）写下你的文章内容。
  4. 保存这个文件。
  5. 之后，当你运行 hexo generate 命令时，Hexo就会自动读取这个 .md 文件，把它里面的Markdown语法转换成标准的HTML网页代码，并最终生成一篇排版精美的博客文章。

总结一个简单的对比：

你以前可能用的方式 Markdown方式
打开Word，点击“标题1”按钮 输入 # 标题
选中文字，点击“加粗”按钮 输入 **加粗文字**
点击“插入图片”按钮 输入 ![](图片地址)
保存为 .doc 文件 保存为 .md 文件

一句话总结：Markdown 就是一种让你“写作时像在聊天，保存后变成 .md 文件，打开后自动排版”的极简技术。 它也是技术圈和写作圈最流行的方式，因为能让你完全专注于内容本身，而不是排版按钮。
